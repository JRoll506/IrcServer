# IrcServer
An Irc server based on RFC 1459. 

Developer: Ryan Bruno <ryanbruno506@gmail.com>
## How To Use
Download and compile source code. Put it separate folder as the jar will make some files. Run using:
```
java -jar <JarName>.jar
```
